Getting Started


                npm install ionic cordova -g
                npm start myApp
                cd myApp
                ionic serve
  • ionic platform add ios, android
  • ionic build ios/android
  • ionic emulate ios
  • from a mac, you'll need xcode, so running ios emulator is easy, but getting android sdk running is slower. I2 recommends using Genymotion
  • Easier just to run ionic serve, open chrome dev tools, and toggle device mode
  • ionic serve --lab: side by side android and ios

Boilerplate


                .
                |-- 
                    |-- bower.json
                    |-- config.xml
                    |-- gulpfile.js
                    |-- hooks
                    |-- ionic.project
                    |-- node_modules
                    |-- package.json
                    |-- platforms
                    |-- plugins
                    |-- resources
                    |-- scss
                    |-- www
  • bower: script dependencies, load ionic code
  • config: for cordova
  • gulpfile: build tool
  • hooks: ignore, cordova
  • ionic.project: ignore, settings file
  • node_modules: local node packages
  • package.json: node dependencies, comes with starter project
  • platform: cordova, native app projects for the platforms you've installed
  • plugins: cordova, cordova stores plugins
  • scss: customize look
  • www: root of app, where you'll do most of your work
